# EcholoN – Enterprise Application Framework for Digital Processes

EcholoN is a configurable Enterprise Application Framework for designing, managing and automating digital business processes. The platform combines workflow design, service management, structured documentation, role-based applications, process templates and integrations in one flexible software environment.

Organizations use EcholoN to build applications for IT Service Management, Enterprise Service Management, customer service, quality management, complaint management, field service, maintenance, self-service portals and individual business workflows.
